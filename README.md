# Jira bot

Написан на TypeScript + node.js  
Использует Jira API, Slack API, Google API  

## Краткое описание

Slack бот, который рассылает изменение в тикетах Jira.  

## Шаги перед запуском

`npm install`  

## Run(development)

`npm start`  

## Run(production)

`npm run build`  
`node prod/index.js`
